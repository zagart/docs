Navigation — Setup Guide

com.zagart.framework.navigation:core:1.0.0

Published via: JitPack · No authentication required

Installation

1. Add JitPack repository

In settings.gradle.kts:

dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
        maven { url = uri("https://jitpack.io") }
    }
}

2. Add dependency

commonMain.dependencies {
    implementation("com.github.zagart.navigation:core:1.0.0")
}
iOS Support: JitPack builds on Linux and cannot produce iOS native binaries (.klib). To use on iOS, include as a Git Submodule or Gradle Composite Build built on macOS.

Quick Start

// Define a destination
data class HomeScreen(override val args: Args = Args())
    : Destination

// Create controller
val controller = rememberNavigationController(
    startDestinations = listOf(HomeScreen())
)

// Wire up the container
NavigationContainer(
    controller = controller,
    scope = {
        screen<HomeScreen> { HomeContent() }
    }
)

FeatureDetails

State persistenceAutomatic save/restore via kotlinx-serialization
Predictive back gestureSupported on Android

Versioning & Releases

Follows Semantic Versioning with phases: alpha, beta, rc, stable.

To release: update version in core/build.gradle.kts, update README, commit, tag, and push. JitPack auto-builds the tag.
