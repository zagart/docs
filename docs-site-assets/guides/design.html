Installation

1. Add JitPack repository

In your root settings.gradle.kts (or root build.gradle.kts):

dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
        maven { url = uri("https://jitpack.io") }
    }
}

2. Add dependencies

In your module's build.gradle.kts:

commonMain.dependencies {
    implementation("com.github.zagart.design:ui:1.6.5")
    implementation("com.github.zagart.design:builder:1.6.5")
}

Versioning & Releases

Follows Semantic Versioning with release phases: alpha, beta, rc, stable.

To release: update version in ui/build.gradle.kts and builder/build.gradle.kts, commit, tag, and push. JitPack auto-builds the tag.
