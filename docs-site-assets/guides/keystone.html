Keystone — Setup Guide

net.zagart.keystone:core:X.Y.Z

Published via: Maven Central · No authentication required

Installation

Add the dependency to your module's build.gradle.kts — Maven Central is already available by default in Gradle projects:

commonMain.dependencies {
    implementation("net.zagart.keystone:core:X.Y.Z")
}

Quick Start

val storage = Keystone.Storage.create(name = "my_app")

// Write
storage.putString("user", "name", "Alice")
storage.putBoolean("user", "notifications", true)

// Read
val name = storage.getString("user", "name", default = "Guest")

// Observe
storage.observe("user") {
    string("name")
    boolean("notifications")
}.collect { snapshot ->
    updateUI(snapshot)
}

Entity Persistence

@Entity(type = "user_profile")
data class UserProfile(
    @Key val id: String,
    val displayName: String,
    @Embedded(prefix = "prefs_")
    val preferences: UserPreferences
)

// Save & load
storage.putEntity(profile)
val loaded = storage.getEntity<UserProfile>("user_123")

Publishing to Maven Central

To publish a new version to Maven Central:

  1. Update version in core/build.gradle.kts
  2. Run ./gradlew :core:publishAllPublicationsToMavenCentral
  3. Tag and push: git tag vX.Y.Z && git push origin vX.Y.Z
